UX Exercise

Please fork this stackblitz before beginning the exercise.

This is open to how creative you want to be.

Create a basic angular application with a masthead, left column navigation and main content area using FLEX.

Must Haves

  • Accessibility: The application needs to be usable by a screen reader;
  • Component for each region in the UX (Masthead/navigation, etc);
  • Good demonstration of your abilities in Angular, JS, HTML and CSS:
    • Component design: Inputs/Outputs, Events;
    • HTML layouts and useage;
    • CSS: Variables, Selectors, Combinators.

Nice To Haves

  • Fixed position masthead and side navigation w/ a scrolling main content area that can be turned on/off by the user;
  • Responsive media queries;
  • SCSS to see how you would structure & re-use your css, variables, etc.

Final Thoughts

Color scheme is up to you, but emphasis on a color palette that is geared towards productivity is preferred.

Measurements are based off of 1rem (16px). Show how you would use margins and paddings in order to minimize white space.
